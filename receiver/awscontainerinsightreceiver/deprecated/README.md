# AWS Container Insights Receiver (deprecated)

This directory contains a deprecated compatibility factory for the legacy
component type `awscontainerinsightreceiver`.

The receiver type has been renamed to `awscontainerinsight`. Please update your
configuration to use `awscontainerinsight` instead of `awscontainerinsightreceiver`.

For full documentation, see the parent component README:
[receiver/awscontainerinsightreceiver/README.md](../README.md)
